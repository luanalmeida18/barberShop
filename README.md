## BarberShop

Projeto desenvolvido como trabalho na UNIVESP. Tendo como objetivo atender a demanda do mercado de barbearias, a fim de facilitar a gestão do negócio. Além de utilizar dos cadastros de clientes para notificar promoções e ofertas disponíveis. Ajudando assim clientes e empresas no convívio profissional.
